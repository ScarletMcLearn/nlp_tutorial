# slackbot(pipモジュール)の利用方法

## インストール
```
# janomeのインストール
$ pip install janome

#slackbotのインストール
$ pip install slackbot
```

## 使用方法
```
$ cd chat/slackbot
$ cp slackbot_settings_sample.py slackbot_setting.py
-> API_TOKENを別途取得したAPI_TOKENに変更
-> PLUGINSから難易度を一つ選択
$ python run.py
```
